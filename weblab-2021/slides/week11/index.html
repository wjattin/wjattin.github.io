CSS Frameworks

A Framework is an application that gives you pre-defined CSS classes, id's and some Javascript functionality to help you speed the process of creating your website without coding everything from scratch.

JQuery is a framework. Javascript is the Language.


Pros

  • Efficiency
  • Security
  • Cross-browser compatibility
  • Good Layouts
  • Cost
  • Support

Cons

  • Learning the framework, not the language
  • You are limited to the Framework's functionality
  • Adds Extra Code
  • Learning curve

Some Popular Html/CSS frameworks

CMS - Content Management Systems

A CMS is a tool that lets you create a website without having to code everything froms scratch.
You can concentrate on the content rather than the coding.

Pros

  • No code experience needed.
  • Many available templates.
  • Faster Development.
  • Many people can contribute content.
  • Easy to use.
  • Pre-developed functionality and features.

Cons

  • Reduced flexibiity in terms of Design.
  • Security Issues if not updated.
  • Uses more server resources.
  • Steeper Learning Curve.
  • Might not be 100% customizable.
