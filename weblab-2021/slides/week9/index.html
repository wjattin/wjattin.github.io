What is Javascript?

Javascript is a scripting language that allows you to add interactivity to web pages.


There are 3 main components in web pages:

HTML

We use it to create the structure of the page.

CSS

We use it to style the elements.

Javascript

We use it to add dynamic functionality interactivity.

Is Javascript a programming language?

  • Javascript is the default scripting language in HTML and is used by all modern browsers.
  • Scripting languages are what can be called a sub-category of programming languages.
  • A Programming language is in simple terms a list of instructions to be performed by the browser or application.
  • Programming Languages: C, C#, C++, Java
  • Scripting Languages: Javascript, PHP, Phyton

Jquery (Write less, do more.)



JQuery is a Javascript Library, not a language.

Libraries are collections of pre-written code you can plug to your webpage without writing full javacript functionality.

Why do we use JQuery?

It's compatible with all browsers.

It's lightweight and fast.

It lets you add complex functionality to your website without having to write full javascript.

It's open source, so anyone can add to it.

It has an extensive plug-in library.

The jQuery library contains the following features:

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX (Asynchronous JavaScript And XML)
  • Utilities
  • Lots of Plugins

How do I add Jquery to my website?

Local Installation

Download the JQuery file and add it to the js folder in your project.

CDN Based Version

Include a link to the file in their Content Delivery Network (CDN).

Javascript / JQuery Glossary

DOM - Document Object Model . The DOM is how the website structure is broken up in a tree hierarchy.

Why it is important to understand the DOM?

Javascript access the DOM and modifies it to create the interaction.

Most of the time it looks just like the html, except on the following cases:

  • It has been modified by a client-side script
  • It has automatically fixed errors in the html
  • There are errors in the source code

Basic Jquery Functions

  • .css();

    Add a css property

    $(".box").css("background-color", "blue");

    .css({"color":"blue","font-size":"30px"});


    Looks for the class .box on the html and adds a background color.

  • .addClass();

    Adds a css Class

    $("#myId").addClass( "yourClass" );

    $("#myId").addClass( "yourClass anotherClass" );


    Looks for the id myId on the html and adds the css class.

  • .removeClass();

    Removes a css Class

    $("#myId").removeClass( "yourClass" );

    $("#myId").removeClass( "yourClass anotherClass" );


    Looks for the id myId on the html and removes the css class.

  • .html();

    Replaces the HTML Content

    $("#myId").html("

    More information

    ");


    Looks for the id myId on the html and replaces the content.

  • Add

    Adds content at the beginning or end of an element

    $("p").prepend("This will go at the beginning");

    $("p").append("This will go at the end");


    Adds content at the before or after an element

    $("p").before("This will go before");

    $("p").before("This will go after");

  • Remove

    Removes the element.

    $(".element").remove();

    Empties the content of the element without removing it.

    $(".element").empty();

Basic Jquery Functions

  • .width();

    Alters the width of an element.

    $("#myDiv").width("100px");

  • .height();

    Alters the height of an element.

    $("#myDiv").height("100px");

Basic Jquery Events

In order for these effects to take place, we need an Event Method to trigger them.
The most common types of events are: Mouse, Keyboard, Form and Document/Window Events.

Mouse Events

  • click();

    Performs an action when clicked

    $( "#target" ).click(function() {
    $("p").hide(); });


    Looks for the id on the html and add the interaction.

  • .dblclick()

    Performs an action when double clicked

    $("#target").dblclick(function(){
    $("p").hide();
    });

  • .mouseenter()

    Performs an action when The mouse enters the area of the selector

    $("#target").mouseenter(function(){
    $("p").hide();
    });

  • .mouseleave()

    Performs an action when The mouse leaves the area of the selector

    $("#target").mouseenter(function(){
    $("p").hide();
    });

  • .hover()

    Performs an action when The mouse hovers over the area of the selector.

    $("#target").hover(function(){
    $(this).css("color","red");
    },
    function(){
    $(this).css("color","black");
    });
    });

Basic Jquery Events

  • The on() Method

    Allows you to attach more than one event handler to an element.
    Specially usefull when you need to restore the element to a previous state after the event is over.


    $("p").on({
    mouseenter: function(){
    $(this).css("background-color", "lightgray");
    },
    mouseleave: function(){
    $(this).css("background-color", "lightblue");
    },
    click: function(){
    $(this).css("background-color", "yellow");
    }
    });

Basic Jquery Effects

  • .show() .hide();

    Shows/Hides an element

    $("#clickBox").show();
    $("#clickBox").hide();


    Looks for the id myId on the html and shows or hides the element.

  • Slide

    Slides an element Up and Down

    $( "#target" ).slideUp(speed, callback);

    $( "#target" ).slideDown(speed, callback);

    $( "#target" ).slideToggle(speed, callback);


    You need an event to trigger the effect.

  • .fadeOut() & .fadeIn()

    Adds a css Class

    $("#myDiv").fadeOut(1000);

    $("#myDiv").fadeIn(1000);


    Looks for the id myId on the html and fades it.

  • .toggle();

    Toggles the action on the element

    $("#toggleBox").toggle(1000);


    Looks for the id myId on the html and toggles the element.

  • .animate();

    Allows you to create custom animations based on most css properties.

    $("button").click(function(){
    $("div").animate({left: '250px', opacity: '0.5'});
    });


    For positioning animation, the element needs to have a position attribute.

    When animating 2 word properties (padding-left) remove the hyphen and use camelCase instead: paddingLeft.
